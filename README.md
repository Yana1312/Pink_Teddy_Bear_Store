Стек:
Visual Studio Insiders - программа с самыми новыми функциями, на которой пишется основной xaml и c# код.

Inkscape - ПО для векторной графики. Использовалось для того, чтобы отобразить элементы в виде Path, когда другие инструменты были для этого непригодны.

Figma - редактор для прототипирования дизайна. 

Docker - платформа для управления приложением в контейнере. Проще говоря, виртуальная машина.

Основная идея: 

* Предоставить пользователям удобный интерфейс для покупки плющевых медвежат.
